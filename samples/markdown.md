# Jellybeans Extended

A dark theme based on [jellybeans.vim](https://github.com/nanotech/jellybeans.vim).

## Features

- Full UI theming for VS Code and Cursor
- Semantic highlighting support
- **Bold text** and *italic text* for emphasis

### Code Examples

Inline code like `const x = 42` stands out clearly.

```python
def hello(name: str) -> str:
    return f"Hello, {name}!"
```

> Blockquotes are styled with a warm tint
> to distinguish them from regular text.
